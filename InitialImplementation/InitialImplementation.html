Initial Implementation

Concept Summary

The interation is the progress bar moving up and down. The user is controlling how fast the bars move, with a slight buffer in between so they can't be pressed without a cooldown in between. The length of the proress bars change over time, as we as the color, changing from red to green as the user interacts with the website.

Primary Variable

let battery = 0


The thing that changes the variable is the keyboard press by the user. and the variable controls the progress and length of the growing bar being controlled by the user's interaction on the website.

Base Build

One Variable: let battery = 0

One Input: up arrow key press

State 1: Bar is red and isn't on screen.

State 2: Bar changes green and length grows to fit canvas size as up arrow is pushed.
