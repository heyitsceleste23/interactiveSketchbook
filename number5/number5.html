Sketch 5: Iteration with an else statement


The way I've been going about this assignment is designing a battery, so my rule I designed was turning the background of the box green when the status is completed.

The otherwise statement makes sure the background stays green when the mouse is clicked enough times, or else it's still in progress, only turning green at the very end.

The user knows the state has changed because the background turns green, and the text is white instead of black, initiating that the state is completed.
